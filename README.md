# 📚 BookLand - Online Book Borrowing Platform

A modern and responsive online book borrowing platform where users can explore books, view details, borrow books, and manage profiles securely.

---

## 🌐 Live Link

https://book-land-fawn.vercel.app/

---

## 🔗 GitHub Repository

https://github.com/SHShadhin/BookLand

---

# 🚀 Features

- User Authentication (Email/Password & Google Login)
- Protected Routes
- Browse All Books
- Search Books by Title
- Category Filtering
- Featured Books Section
- Single Book Details Page
- Borrow Book Feature
- User Profile Page
- Update User Information
- Responsive Design
- Toast Notifications
- Mobile Friendly Navbar


---

# 🛠️ Technologies Used

- Next.js
- React.js
- Tailwind CSS
- DaisyUI
- HeroUI
- BetterAuth
- MongoDB
- React Toastify
- React Icons
- Animate.css

---

# 📂 Pages

| Route | Description |
|---|---|
| / | Home Page |
| /all-books | All Books Page |
| /books/[id] | Book Details Page |
| /login | Login Page |
| /signup | Registration Page |
| /profile | User Profile |
| /update-profile | Update User Information |
